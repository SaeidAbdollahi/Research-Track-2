Kinematic Monitor Node

platform

Unix

synopsis

Python module for monitoring the odometry data.

Detailed Description:

This node subscribes to the ‘odometer’ topic to receive the custom OdoSensor message, also it prints the ‘distance to the target’ and ‘average speed’ of the robot. It publishes these data as a custom ‘KinematicData’ message

Communication Protocole:

It uses the publish-subscribe protocole to transfer the odometry messages.

Goal Summary Node

platform

Unix

synopsis

Python module for monitoring the goals status.

Detailed Description:

This node subscribes to the ‘reaching_goal/feedback’ topic to check the current goal status, then it creates a service to print the number of reached or cancelled goals.

Communication Protocole:

It uses the client-service protocole for communication.
